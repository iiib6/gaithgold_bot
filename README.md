# 🚀 بوت واتساب الغيث للذهب والمجوهرات & لوحة التحكم

نظام متكامل ومبتكر لمكائن محلات صياغة وبيع الذهب والمجوهرات (باسم **الغيث**)، يدمج بين حاسبة ذهب رسومية تفاعلية، ومحاكي محادثات واتساب ذكي، بالإضافة إلى محرك اتصال حقيقي بالواتساب (WhatsApp Bot) يتيح تعديل الهامش وتحديث الأسعار لجميع الزبائن تلقائياً بمجرد إرسال رسالة من هاتف المالك!

---

## ✨ المميزات الرئيسية
1. **لوحة تحكم وتعديل حي**: إدارة سعر الأونصة العالمي وسعر الصرف بالدينار العراقي وقيمة هامش السوق من واجهة واحدة فاخرة تدعم التأثيرات الحركية والتصميم الزجاجي الفخم.
2. **حساب تلقائي دقيق للعيارات**: حساب فوري لأسعار عيارات الذهب (24، 21، 18) للغرام والمثقال العراقي (المثقال = 5 غرام) بالتناسب الرياضي الدقيق.
3. **جلب سعر الأونصة لحظياً**: زر تحديث يجلب السعر الفعلي لأونصة الذهب في البورصة العالمية مباشرة عبر الـ API المجاني الموثوق (`gold-api.com`).
4. **محاكي محادثات مدمج**: واجهة تحاكي هاتف واتساب بالكامل تتيح لك تجربة محادثة الزبون مع البوت، مع إمكانية تبديل هوية المرسل (زبون / المالك) لاختبار البوت وتجربة كافة الميزات من المتصفح مباشرة.
5. **ربط واتساب الحقيقي**: البوت مبني ليدعم الربط برقم واتساب حقيقي عبر رمز الـ QR (باستخدام مكتبة Baileys المستقرة والخفيفة التي لا تتطلب متصفح Chromium ثقيل).
6. **سجل عمليات حي (Terminal Logs)**: مراقبة فورية لجميع الطلبات وحركات رسائل الواتساب الواردة والصادرة على جانب اللوحة.

## 📐 نظام الحسابات الرياضية المعتمد
* **غرام 24** = (سعر الأونصة العالمي × سعر الصرف د.ع/$) ÷ 31.1035
* **غرام 21** = غرام 24 × (21 ÷ 24)
* **مثقال 21 الأساسي** = غرام 21 × 5
* **مثقال 21 المعدل** = مثقال 21 الأساسي + **هامش السوق**
* **غرام 21 المعدل** = مثقال 21 المعدل ÷ 5
* **غرام 24 المعدل** = غرام 21 المعدل × (24 ÷ 21)
* **مثقال 24 المعدل** = غرام 24 المعدل × 5
* **غرام 18 المعدل** = غرام 24 المعدل × (18 ÷ 24)
* **مثقال 18 المعدل** = غرام 18 المعدل × 5

## 🛠️ طريقة التشغيل

### الخطوة 1: تثبيت الاعتماديات الأساسية للوحة والمحاكي
افتح موجه الأوامر (Terminal) في مجلد المشروع، وشغّل الأمر التالي:
```bash
npm install
```

### الخطوة 2: تشغيل الخادم
لتشغيل الخادم المحلي على جهازك:
```bash
npm start
```
سيظهر لك في سطر الأوامر أن الخادم يعمل على المنفذ 3000:
`🚀 Al-Ghaith Gold Bot Server is running on port 3000`
`📂 Web Dashboard available at http://localhost:3000`

افتح الرابط التالي في المتصفح لرؤية لوحة التحكم والمحاكي:
👉 **[http://localhost:3000](http://localhost:3000)**

## 🔗 ربط رقم واتساب حقيقي (اختياري)
في حال كنت ترغب في تشغيل البوت على هاتف المحل الحقيقي ليرد على الزبائن الفعليين بالأسعار:

### 1. تثبيت مكتبة الاتصال بالواتساب
أوقف الخادم أولاً بالضغط على `Ctrl + C` ثم ثبّت مكتبة الاتصال المخصصة للواتساب:
```bash
npm install @whiskeysockets/baileys qrcode
```

### 2. تشغيل الخادم مجدداً
```bash
npm start
```
ستلاحظ في اللوحة أن زر **بدء الربط بالواتساب** أصبح مفعلاً.

### 3. مسح الرمز QR
* انتقل إلى تبويب **ربط واتساب الحقيقي** في لوحة التحكم.
* اضغط على زر **بدء الربط بالواتساب**.
* انتظر ثوانٍ وسيظهر رمز الاستجابة السريعة (QR Code) على الشاشة.
* افتح واتساب في هاتف المحل، اذهب إلى **الأجهزة المرتبطة** -> **ربط جهاز** ثم امسح الرمز الظاهر على الشاشة.
* بمجرد إتمام المسح، سيتصل البوت تلقائياً وتتغير الأيقونة إلى اللون الأخضر (متصل بنشاط).

## 💬 كيفية عمل أوامر التحكم بالواتساب (أمر تعديل الهامش)
عند اتصال رقم المالك بالبوت، يمكن لمالك المحل التحكم بالأسعار وإعادة ضبطها عن بُعد وبدون الحاجة لفتح المتصفح:
1. يرسل المالك (من رقمه الذي تم تكوينه في لوحة التحكم) رسالة واتساب إلى رقم البوت بالصيغة:
   * `تعديل +3000` (لإضافة هامش قدره 3000 دينار عراقي على سعر المثقال 21).
   * `تعديل -5000` (لخصم هامش قدره 5000 دينار عراقي).
2. سيقوم البوت بقراءة الرسالة، وتحديث ملف الإعدادات `config.json` محلياً، وإعادة حساب جميع الأسعار تلقائياً.
3. يرسل البوت رسالة تأكيد للمالك تحتوي على تأكيد التعديل والأسعار الجديدة فوراً.
4. أي زبون يسأل البوت بعد ذلك عن السعر (بإرسال رسائل مثل: "بيش المثقال اليوم؟" أو "سعر الذهب الحالي") سيتلقى الأسعار الجديدة المعدلة تلقائياً.

## 📁 هيكلية ملفات المشروع
* `server.js`: خادم Express المسؤول عن واجهات البرمجة وإدارة الحسابات وتكامل واتساب.
* `config.json`: ملف حفظ التكوين المستمر (سعر الأونصة، سعر الصرف، الهامش، رقم المالك).
* `package.json`: ملف تعريف الاعتماديات وإدارة الحزم.
* `public/index.html`: الواجهة الرسومية الأنيقة للوحة التحكم والتسعير والمحاكاة.
* `public/style.css`: التنسيقات الفاخرة للواجهة (glassmorphism وتأثيرات ذهبية متناسقة).
* `public/app.js`: منطق التفاعل في الواجهة والمحاكاة ومعالجة الاتصال بالخادم.
